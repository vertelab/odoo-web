Tillgänglighetsredogörelse för Kundrelationssystemet

Arbetsförmedlingen är ägare till webbplatsen för Kundrelationssystemet (CRM) och vi vill att alla ska kunna använda webbplatsen, oavsett behov. Här redogör vi hur webbplatsen uppfyller webbriktlinjerna om tillgänglighet, eventuella kända tillgänglighetsproblem och hur du kan rapportera brister till oss så att vi kan åtgärda dem.

Hur tillgänglig är webbplatsen?

Vi är medvetna om att delar av webbplatsen inte är helt tillgängliga. I avsnittet om innehåll som inte är tillgänglig listar vi kända brister i tillgänglighet.

Kontakta oss om du hittar fler brister

Vi strävar hela tiden efter att förbättra webbplatsens tillgänglighet. Om du upptäcker problem som inte är beskrivna på den här sidan, eller om du anser att vi inte uppfyller lagens krav, meddela oss så att vi får veta att problemet finns.

Teknisk information om webbplatsens tillgänglighet

Den här webbplatsen är inte förenlig med webbriktlinjerna om tillgänglighet.

Innehåll som inte är tillgängligt

Vi är medvetna om följande brister som inte följer lagkraven:

Problem vid användning utan syn

  • Det är inte möjligt för användare som navigerar med skärmläsare att uppfatta informationsområden på sidorna i systemet, eftersom det saknas landmärken för informationsområden i koden.

  • Identifiering av objekt och gränssnittskontroller är inte konsekvent med andra system på myndigheten eftersom den plattform som Kundrelationssystemet saknar stöd för komponenter i myndighetens designsystem.

  • Hänvisningar till komponenter och innehåll baseras i vissa fall på sensoriska kännetecken som form, storlek, placering eller färg.

  • Tabb- och fokusordning vid tangentbordsnavigation är inte alltid logisk.

Problem vid användning med nedsatt syn

  • Det finns vissa problem med att ändra textstorleken i gränssnittet eftersom exempelvis menyer inte fungerar med större text än 175 %.

  • Det är inte möjligt för användaren att justera avstånd i text.

  • Vid fokus på vissa objekt saknas en visuell fokusmarkör som visar vilket objekt som har fokus vid tangentbordsnavigation.

Problem vid användning med begränsad rörlighet

  • Gränssnittet kan inte styras med tangentbord eller med hjälpmedel som bygger på liknande mekanismer som tangentbordsstyrning.

  • Vid fokus på vissa objekt saknas en visuell fokusmarkör som visar vilket objekt som har fokus vid tangentbordsnavigation.

  • Tabbordning vid tangentbordsnavigation är inte alltid logisk.

Problem vid användning med kognitiv nedsättning

  • Sortering och placering av objekt i navigationsområden är inte konsekvent med andra system på myndigheten eftersom den plattform som Kundrelationssystemet saknar stöd för komponenter i myndighetens designsystem.

  • Hänvisningar till komponenter och innehåll baseras i vissa fall på sensoriska kännetecken som form, storlek, placering eller färg.

  • Det är inte möjligt för användaren att justera avstånd i text.

Hur vi jobbar med digital tillgänglighet

Vi strävar efter att Arbetsförmedlingens webbplatser ska kunna uppfattas, hanteras och förstås av alla användare och medarbetare, oavsett behov eller funktionsnedsättning och oberoende av vilka hjälpmedel du använder. Vi ska åtminstone uppnå grundläggande tillgänglighet genom att följa WCAG 2.1 på nivå AA.

Ett arbete pågår för att se över Arbetsförmedlingens digitala riktlinjer. För att ge våra medarbetare rätt förutsättningar och kunskap om digital tillgänglighet lägger vi stort fokus på att utbilda dem som utvecklar våra digitala tjänster.

Ansvaret att tillgänglighetskraven följs ligger framförallt hos sektionens produktägare. Till sin hjälp har produktägaren UX-ingenjörer, vars uppgift är att vidareutveckla, testa och genomföra användarstudier för att uppfylla tillgänglighetskraven enligt webbriktlinjerna. Arbetet sker kontinuerligt och parallellt med utvecklingen av systemet.

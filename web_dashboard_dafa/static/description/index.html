Features


This module adds personal dashboard for all users. Also allow to edit(configure) it who has access of 'My Dashboard Manager'.

Setup-instructions


Go to Settings >> User & companies >> Users. Give acess of 'My Dashboard Manager' group to users for who you want allow edit My Dashboard view.

User-guide


- Adds a group 'My Dashboard Manager'.

- Adds menu 'My Dashboard' for all users. By default after installing module it shows following scrren with user name and Welcome message.
- 'My Dashboard Configuration'.
- Here you can see and edit dashboard view for all users.

- Here you can change Welcome message and its description.
- Also configure boxes. For e.g. Here Added one box for tasks and to redirect user to tasks view directly from here.
- In boxes you can configure Title, Sub-title, Description etc. In Actions, You need to choose action on which you want to redirect user. In Image class, need to add Font Awesome icons class to show image. In Groups, need to add groups, like for if user has this group then show this box otherwise not.
- Here we have another menu inside configuration Update Dashboard Data to configure dashboard at a time for more than one user.
- It is same like we configures for each user, Just added one flag 'Want to update Welcome Data?' If this flag set then update Welcome msg and its description otherwise update only boxes. Also here need to select Users at last. So this box adds in selected user's dashboard only.
